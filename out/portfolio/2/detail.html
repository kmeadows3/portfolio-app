Chatwick

Capstone project for Tech Elevator. Worked in a small team using Agile methodologies to build a rules-based chatbot designed to help prepare Tech Elevator students search for the job search and interview process. A Vue.js front-end sends user utterances to a RESTful Java application using Spring Boot to run a Tomcat server. Chatbot responses are retrieved from a PostgreSQL database, then filtered and ranked to determine the most fitting response. It is capable of providing general interview advice, providing practice HR and technical questions, generating and scoring a quiz on technical topics, and even searching for jobs.
Java, PostgreSQL, Vue.js, JavaScript, HTML/CSS
